# Código de Conduta

Queremos que a colaboração neste repositório seja respeitosa, inclusiva e
produtiva para todo mundo.

## Escopo

Este código de conduta se aplica a:

- issues, pull requests e discussões neste repositório
- interações entre colaboradores em espaços diretamente ligados a este fork

Ele é aplicado pelos mantenedores deste repositório. Este documento não
representa moderação oficial do Rust Project nem da Rust Foundation.

## Comportamento esperado

- trate outras pessoas com respeito e boa-fe
- faça crítica técnica sem ataque pessoal
- explique contexto ao discordar de uma tradução ou decisão editorial
- esteja aberto a revisão, correção e aprendizado coletivo

## Comportamento inaceitavel

- assédio, intimidação ou perseguição
- insultos, ataques pessoais ou hostilidade recorrente
- publicação de informações privadas sem permissão
- linguagem discriminatória ou excludente
- comportamento que torne a colaboração insegura ou inviável

## Como relatar

O canal inicial de relato é por issues públicas neste repositório.

Se o caso for sensível, você pode abrir uma issue curta pedindo um canal
privado com os mantenedores, sem expor detalhes publicamente.

## Aplicação

Os mantenedores podem editar, ocultar, trancar ou remover interações e
contribuições que violem este código, além de restringir participação quando
necessário para proteger a comunidade.
